# HN 评分偏好

## 核心判断维度
(人类初稿, evolve 后会被重写)
- HN 的价值在讨论热度 >= 单纯 score, 评论数 100+ 的优先
- 优先: 开源 agent/框架发布, MCP 类工具, 技术深度文章, 作者历史高质量
- 次优: benchmark 对比, 技术综述, 官方 release 说明
- 扣分: 已被硬规则漏过的融资/产业/政策/硬件业
- "Show HN" 类型若是完整开源项目加分, 若是 wip demo 减分
